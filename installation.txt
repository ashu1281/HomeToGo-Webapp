# HomeToGo-Webapp
# This Web application helps user to register Room's to assist you in finding the perfect rental home or room. Also you can find your next rental from search view in your targeted area.
# Built with Core PHP, PHP PDO, Bootstrap4, MYSQL.

installation of house rental and property listing project(HomeToGo)


Softwares need 

1. xamp or wamp 

2. xamp/wamp - extract the zip file to inside htdocs/www.

3. start xamp/wamp server.

4. open browser and type - localhost/HomeToGo


show error--

SQLSTATE[HY000] [1049] Unknown database 'newrent'


5. goto localhost/phpmyadmin .

and create database newrent

and import newrent.sql file

6. finally project is successfully installed
